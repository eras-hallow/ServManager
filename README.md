FR

**ServManager** est un client RCON développé en Python

## Installation

1. Téléchargez l'archive ZIP.
2. Extrayez le contenu.
3. Exécutez `servman.exe` pour lancer le programme.

## Utilisation

- Utilisez le fichier `servman.py` pour voir le code source et personnaliser le script.
- Le fichier `servman.spec` peut être utilisé pour recréer l'exécutable avec PyInstaller.

EN

**ServManager** is an RCON client developed in Python

## Installation

1. Download the ZIP archive.
2. Extract the contents.
3. Run `servman.exe` to launch the program.

## Usage

- Use the `servman.py` file to view the source code and customize the script.
- The `servman.spec` file can be used to rebuild the executable with PyInstaller.
